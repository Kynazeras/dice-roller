# Dice Roller
This is a game I am using to learn some basic game mechanic implementations in Godot. Right now all it really does is allow you to roll dice and try to get exactly as your total. 
Dice Roller uses a data-driven design to keep dice results clean and readable across the applcation.

# Dicier Font
If you want the full experience, you will need to download the [Dicier Font](https://speakthesky.itch.io/typeface-dicier) from SpeakTheSky on itch.io. It is free to download and you can make a financial contribution if you want.
